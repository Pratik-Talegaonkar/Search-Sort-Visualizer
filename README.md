## üßÆ Implemented Algorithms

### Sorting
1.  **Bubble Sort**: The classic stepping stone. Simple but effective for learning.
2.  **Selection Sort**: Visualizes how the minimum element is selected and placed.
3.  **Insertion Sort**: Shows how an array is built one item at a time.
4.  **Quick Sort**: Demonstrates the power of Divide and Conquer with partitioning.
5.  **Merge Sort**: visualization of splitting and merging arrays.

### Searching
1.  **Linear Search**: A simple sequential search.
2.  **Binary Search**: An optimized search for sorted arrays (includes auto-sorting visualization!).
